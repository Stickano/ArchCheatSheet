===== Usefull Packages =====

== Thermald ==
Helps prevent overheating situations.
Install the //thermald// (AUR) package

== i7z ==
Reporting tool for Intel processors. Can be invoked with //i7z// or //i7z-gui//
Install the i7z package

== CpuPower ==
Assists with CPU frequency scaling. 
Install the //cpupower// package
Enable the //cpupower.service //file
Edit 	///etc/default/cpupower//	change to 	**governor='performance'**
