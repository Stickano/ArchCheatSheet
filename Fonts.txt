===== Fontconfig =====
**# Fontconfig is a default library used by many linux distros. **

**# You can configure Fontconfig by picking presets in **///etc/fonts/conf.avail///
**# and Symlinking them to **//~/.config/fontconfig/conf.d///
**# Ex.**
$ cd ~/.config/fontconfig/conf.d/
$ ln -s /etc/fonts/conf.avail/10-sub-pixel-rgb.conf

**# The same rules defined in **///etc/fonts/conf.avail///
**# can also be defined in a user-configuration **//~/.config/fontconfig/fonts.conf//
**# User-defined configuration at Github.**
**# When using the **//fonts.conf//** you should Symlink to **//50-user.conf//
$ cd ~/.config/fontconfig/conf.d/
$ ln -s /etc/fonts/conf.avail/50-user.conf

**# So far, the best solution I've found, is to;**
**# Disable **//autohint//
**# Enable **//hinter//** (actually think this is default - but hey)**
**# Set **//hintstyle//** to **//hintslight//
**# Enable **//subpixel-rgb //**(RGB monitor)**
**# When **//subpixel-rgb//** is defined, also enable **//LCD-filter//**; **
**# The **//lcddefault//** seems to work fine.**


== TeX live packages ==
**# By default, TeX packages are not read by Fontconfig. This will make some text render wrong/bad.**
**# Correct this by installing TeX-live packages and Symlinking it to Fontconfig.**
$ pacman -S texlive-core 

**# Symlink the location**
$ ln -s /etc/fonts/conf.avail/09-texlive-fonts.conf /etc/fonts/conf.d/09-texlive-fonts.conf

**# Then update Fontconfig**
$ fc-cache && mkfontscale && mkfontdir


===== Infinality =====
**# Infinality comes with a great deal of improvements for font rendering. **

== Installation ==
**# To install Infinality, you first have to add a couple of repositories to your **///etc/pacman.conf//
{{{code: lang="texinfo" linenumbers="True"
# 1; The bundle 
[infinality-bundle]
Server = http://bohoomil.com/repo/$arch

# 2; Multilib
[infinality-bundle-multilib]
Server = http://bohoomil.com/repo/multilib/$arch

# 3; Fonts
[infinality-bundle-fonts]
Server = http://bohoomil.com/repo/fonts
}}}


**# You also have to import the maintainers key (**//962DDE58//**)**
$ pacman-key -r 962DDE58
$ pacman-key -f 962DDE58
$ pacman-key --lsign-key 962DDE58

**# Then update your system**
$ pacman -Syyu

**# And install core packages + multilib**
**# There will be conflicts! **//Yes//** them all!**
$ pacman -S infinality-bundle infinality-bundle-multilib ibfonts-meta-extended

**# Whenever you find a sweet font - always check if it's in the **//Infinality-bundle-fonts//** repository**
**# Just look through the fonts from that repository first, when you package search a font **
$ pacman -Ss //<FONT_NAME>//

**# Recommended patches to install**
$ yaourt -S freetype2-infinality fontconfig-infinality

== Configuration ==
**# By default, Infinality ships with a **//Ultimate3//** setting, which works fine.**
**# I find the **//Ultimate4//** to be just a little sweeter though. **
**# **//Ultimate4//** will render the text a little darker/thicker.**

**# To define this setting, a file must be copied and edited**
$ cp /usr/share/doc/freetype2-infinality-ultimate/infinality-settings.sh /etc/X11/xinit/xinitrc.d/

**# Then edit and outcomment the **//INFINALITY_FT="ultimate3" //**and change to **//ultimate4//
