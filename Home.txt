===== Base Installation and Configuration =====
**# Follow the order to a stable base setup**
**1 )** [[Arch_Base]] (Arch installation)
**2 ) Check out **//netctl//** in **//Network//** for wifi setup and auto connection **
**3 ) Check out **//Users// //and Groups//** in **//Users//** to create a user**
**4 ) **[[sudo]] 
**5 ) Enable 32bit support from **//Mirros//** in**// Maintenance //
**6 )** [[Xorg]] (+xinit) (Display Server)
**7 ) **[[Touchpad]] 
**8 )** [[XDM]] (Display manager/login prompt)
**9 )** [[Kernels]] (zen & lts)
**10 ) **[[DKMS]] 
**11 )** [[Display_Driver]] (intel/nvidia with bumblebee)
**12 )** [[AUR_Support]] (yaourt)
**13 ) **[[i3]] (window manager)
