== Laptop Mode Tools ==
Controls for many power-saving features. 
Install the //laptop-mode-tools// (AUR) package

Main configuration file is located at ///etc/laptop-mode.conf//	
Configuration files are located at 	///etc/laptopmode/conf.d///

**Check available modules and their status : **
$ grep -r '^\(CONTROL\|ENABLE\)_' /etc/laptop-mode/conf.d

**Enable ** //laptop-mode.service	//
