**You can use the lts kernel as fallback, should your system encounter a critical/kernel error.** 

**# Install it from the official repository**
$ pacman -S linux-lts 

== A little additional configuration ==
**# Grub, by default, groups the **//linux//** and **//linux-lts//** kernel in a category tree (subcategory).** 
**# To disable this, edit** ///etc/default/grub //**and add the following line** 
{{{code: lang="texinfo" linenumbers="True"
GRUB_DISABLE_SUBMENU=y
}}}


**# Update the bootloader with ** 
$ grub-mkconfig -o /boot/grub/grub.cfg
