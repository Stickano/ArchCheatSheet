====== xdg-utils ======
Created s√∏ndag 04 december 2016

**# **//xdg-utils//** will let you set, and see, the default application which handles different filetypes (mime-types)**
**# **//xdg-utils//** should already be installed on your system.**

== Get mime-type ==
**# We'll use a **//.conky//** file as example**
$ xdg-mime query filetype .conkyrc

**# Should return something like **//plain/text//

== Get default application for mime-type ==
$ xdg-mime query default plain/text

== Change default application ==
**# You can see all your installed, useable, applications in **///usr/share/applications///

**# To change the default application for a mime-type**
$ xdg-mime default gedit.desktop plain/text
